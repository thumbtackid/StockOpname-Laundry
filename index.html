Penjelasan Parameter

  • Bed Rawat Inap: Jumlah total bed yang tersedia.
  • BOR (Bed Occupancy Rate): Persentase keterisian rata-rata rumah sakit (default 70%).
  • Operasi / Hari: Estimasi rata-rata operasi per hari untuk kebutuhan set operasi.
  • Buffer: Jumlah siklus laundry cadangan (Safety Stock). Default 3 siklus.

Stock Opname (Penghitungan Fisik)

1. Input jumlah barang fisik yang ditemukan.
2. Jika ada selisih (kurang/rusak/hilang), sistem akan menandainya.
3. Klik "Finalisasi & Update Stok" untuk menjadikan angka fisik ini sebagai stok master baru.
